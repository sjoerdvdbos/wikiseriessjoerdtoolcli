============
Installation
============

At the command line::

    $ pip install wikiseriessjoerdtoolcli

Or, if you have virtualenvwrapper installed::

    $ mkvirtualenv wikiseriessjoerdtoolcli
    $ pip install wikiseriessjoerdtoolcli

Or, if you are using pipenv::

    $ pipenv install wikiseriessjoerdtoolcli
Or, if you are using pipx::

    $ pipx install wikiseriessjoerdtoolcli
